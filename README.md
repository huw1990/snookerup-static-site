# SnookerUp Static Site

This project is for coming up with the general style of the pages of the SnookerUp website. When all necessary pages are complete, they will form the basis of the pages of the SnookerUp frontend (written in React).

## Tailwind Setup

This project uses Tailwind for CSS.

In development, the following command should be run for generating the output CSS files.

`npx tailwindcss -i css/input.css -o css/styles.css --watch`
